# Syntax for Github Pages Markdown

Not covering markdown stuff, but how Github contorts markdown into doing website things. For regular Markdown syntax, visit: [This page](google.com/search?q=github+flavored+markdown)



In Markdown Files (.md)
Use Markdown's link syntax [Link Text](URL). Again, use relative paths.
For a page in the same directory:
markdown
[About Me](about.md)
For a page in a subdirectory:
markdown
[Contact](pages/contact.md)
For a page in the parent directory:
markdown
[Home](../index.md)
